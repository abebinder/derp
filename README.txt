This directory contains the documentation and source code for DERP. DERP is an operational support system for the winter 2017 delivery of CIS322 and is intended to aid in the management of status reports and solution testing. More information can be found in the product requirements and software design document (documentation/derp.pdf).

Directory contents:
README.txt      - This readme document
INSTALL.txt     - Installation instructions
documentation   - LaTeX build files for the business style documentation
sql             - Scripts related to setup and management of the DERP database
src             - Source code for the web application
